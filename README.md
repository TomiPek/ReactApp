React appi

Tämä react appi hakee https://mernprojekti3.herokuapp.com/ projektin dataa ja käyttää niitä muodostamaan niistä tiedoista material tablen, jossa on lisää, poista, update ja get tyyppisiä tietoja. Projektissa on importattu material-table https://material-table.com/#/ osoitteesta ja bootstrap-react komponentteja ja react material ui sivustolta buttonit. Projektia ei ole tässä vaiheessa julkaistu mihinkään sivustolle.
